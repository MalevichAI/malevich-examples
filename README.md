# Malevich Examples

Here you can find examples of flows that can be created with Malevich. You can use them as a starting point for your own flows or
just to get a better understanding of how Malevich works. We are constantly adding new examples, so stay tuned!

## Available examples

### Duolingo

This example is a multi-modal flow with moderate complexity and shows the power of Malevich. Checkout the [Duolingo](https://github.com/malevichAI/malevich-examples/tree/master/duolingo) example.
